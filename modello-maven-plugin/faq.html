Is Modello really used anywhere?

Modello is used extensively in Maven. You can find examples of models:

  • Maven's POM: you can see the mdo file for this model
  • Maven's settings: you can see the mdo file for this model
  • Maven assembly plugin: uses both a descriptor model and a component model.
  • Doxia Decoration model, aka site.xml model.
